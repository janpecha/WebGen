Install
=======

1. Open file ```webgen``` in a text editor
2. Change path to WenGen directory (```/path/to/webgen.php```) on last line of the file
3. Save the file
4. Copy the file ```webgen``` into directory which is located in ```PATH``` (```~/bin/``` on Ubuntu system)


Usage
=====

```
cd MyWeb
webgen --run
```
